# Project Overview

The User Management System is a full-stack backend application developed using Node.js, Express.js, and MySQL.

The main purpose of this project is to build a RESTful API that manages user records with proper database integration, validation, and security practices.

This project supports complete CRUD functionality:
- Creating new users
- Fetching user details
- Updating existing user information
- Deleting user records
